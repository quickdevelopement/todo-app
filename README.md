* **Frontend**: HTML5, CSS4, JavaScript (ES6+), React.js

## Installation & Setup
Follow these quick setup instructions to run the application locally on your Computer

### Prerequisites
Make sure you have Node.js and npm intalled on your machine.

### 1. Clone the Repository
```bash
git clone https://github.com/quickdevelopement/todo-app.git
cd todo-app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variable(Optional)
Create a `.env` file in the root directory and add your configureations:
```env
PORT=3000
DB_URI=your_db_connections_string
```

### 4. Run the Application
Start the local development Server:
```bash
npm start
```
Open your browser and navigate to `http://localhost:3000` to interact with the application

## Running Tests
To execute the test suites built for this application, run:
```bash
npm test
```
